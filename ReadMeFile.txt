*/

1) Make a database (like - lee_and_nee_db ) in your phpmyadmin and import the database file you have received in mail.

-------------------------------------------------

2) DATABASE CONNECTIVITY SETTINGS

Change the database username , password and database name in database.php as per your database configuration.
Path is given below -

lee_and_nee_assignment\application\config\database.php
-------------------------------------------------

3) Base Site URL SETTINGS (Applicable only if you change the current base url setting)

If required you can change the base url setting in config.php page like 
$config['base_url'] = "http://".$_SERVER['HTTP_HOST']."/lee_and_nee_assignment/";

Path is given below -
lee_and_nee_assignment\application\config\config.php
-------------------------------------------------
